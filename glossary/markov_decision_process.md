# Markov Decision Process (กระบวนการตัดสินใจแบบมาร์คอฟ)

[← กลับสู่เนื้อหาบทเรียน](../lectures/project3_agentic_workflows_tools.md)

**Markov Decision Process (MDP)** หรือกระบวนการตัดสินใจแบบมาร์คอฟ เป็นกรอบแนวคิดทางคณิตศาสตร์ (Mathematical Framework) สำหรับใช้สร้างแบบจำลองการตัดสินใจ ในสถานการณ์ที่ผลลัพธ์ของการกระทำ (Action) มีทั้งส่วนที่ควบคุมได้ (Deterministic) และส่วนที่เป็นการสุ่ม (Stochastic) 

เป็นรากฐานที่สำคัญที่สุดของวิชา Reinforcement Learning (RL) และระบบ AI Agent ขั้นสูง

## คำอธิบายเชิงลึก (In-depth Explanation)
ระบบ MDP จะถูกนิยามด้วย Tuple ประกอบด้วยองค์ประกอบ 5 อย่าง $(S, A, P, R, \gamma)$:
1.  **$S$ (State Space)**: เซตของสถานะ (State) ที่เป็นไปได้ทั้งหมดของสิ่งแวดล้อม
2.  **$A$ (Action Space)**: เซตของการกระทำที่เอเจนต์สามารถเลือกทำได้
3.  **$P$ (Transition Probability: $P_a(s, s')$)**: ความน่าจะเป็นที่ถ้าเอเจนต์อยู่ที่สถานะ $s$ แล้วเลือกกระทำ $a$ สิ่งแวดล้อมจะเปลี่ยนไปยังสถานะใหม่ $s'$
4.  **$R$ (Reward Function: $R_a(s, s')$)**: ผลตอบแทนหรือรางวัลที่เอเจนต์จะได้รับทันทีเมื่อเปลี่ยนจาก $s$ ไป $s'$ ด้วยการกระทำ $a$
5.  **$\gamma$ (Discount Factor)**: ตัวคูณลดทอน (ค่าระหว่าง 0 ถึง 1) ใช้สำหรับกำหนดความสำคัญของรางวัลในอนาคตเทียบกับรางวัลในปัจจุบัน

**คุณสมบัติมาร์คอฟ (Markov Property)** คือหัวใจของระบบนี้: "อนาคตขึ้นอยู่กับสถานะในปัจจุบันเท่านั้น ไม่ได้ขึ้นอยู่กับอดีต" (The future is independent of the past given the present) หมายความว่าเราไม่จำเป็นต้องรู้ว่าเอเจนต์เดินผ่านอะไรมาบ้างในอดีต ถ้ารู้เพียงสถานะปัจจุบัน ณ วินาทีนี้ ก็เพียงพอแล้วที่จะคำนวณการตัดสินใจต่อไป

เป้าหมายของ Agent ในระบบ MDP คือการหานโยบาย (Policy: $\pi$) ที่ดีที่สุด ที่ช่วยให้ได้รับผลตอบแทนรวมระยะยาว (Cumulative Expected Reward) สูงสุด

## ตัวอย่างการนำไปใช้ (Practical Example)
*   **หมากรุก / AlphaGo**: 
    *   State = ตำแหน่งหมากบนกระดานปัจจุบัน
    *   Action = การเดินหมากตาต่อไป
    *   Reward = 0 ระหว่างเกม และ +1 เมื่อชนะ -1 เมื่อแพ้
    *   Transition = การเดินหมากของคู่แข่ง (ซึ่งเราควบคุมไม่ได้)
*   **LLM Agent ท่องเว็บ**: 
    *   State = หน้าจอเว็บเบราว์เซอร์ปัจจุบัน 
    *   Action = การคลิกปุ่มหรือพิมพ์ข้อความ
    *   Reward = 1 หากได้ข้อมูลที่ต้องการตามคำสั่งผู้ใช้
