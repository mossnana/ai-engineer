# System 1 vs System 2 Thinking (ระบบการคิด 1 และ 2)

[← กลับสู่เนื้อหาบทเรียน](../lectures/project4_deep_research_reasoning.md)

**System 1 และ System 2 Thinking** เป็นทฤษฎีทางจิตวิทยาการรู้คิด (Cognitive Psychology) ที่ถูกนำเสนอโดยนักเศรษฐศาสตร์รางวัลโนเบล Daniel Kahneman ในหนังสือเรื่อง _Thinking, Fast and Slow_ เพื่ออธิบายกลไกการทำงานของสมองมนุษย์ออกเป็นสองระบบ

ในยุคปัจจุบัน แนวคิดนี้ถูกนำมาใช้อธิบายระดับสติปัญญาและการออกแบบสถาปัตยกรรมของปัญญาประดิษฐ์ (AI) อย่างแพร่หลาย โดยเฉพาะในการเปรียบเทียบระหว่าง LLM รุ่นปกติ (GPT-3/4) กับโมเดลเน้นการใช้เหตุผล (OpenAI o1)

## คำอธิบายเชิงลึก (In-depth Explanation)

- **System 1 (Thinking Fast)**:
  - **ลักษณะ**: รวดเร็ว, อัตโนมัติ, ใช้สัญชาตญาณ, ไม่ต้องใช้ความพยายาม, ทำงานอยู่ตลอดเวลา
  - **ตัวอย่างในมนุษย์**: การตอบว่า 2+2 เท่ากับเท่าไร, การอ่านป้ายโฆษณา, การขับรถในเส้นทางที่คุ้นเคย
  - **ตัวอย่างใน AI**: LLM แบบดั้งเดิม (เช่น ChatGPT สมัยก่อน) ที่ใช้กลไก Autoregressive ทำนายคำถัดไปทันทีผ่าน Forward Pass เพียงรอบเดียว มันจะตอบคำถามได้รวดเร็วเหมือนใช้สัญชาตญาณ แต่ก็มักจะ "หลอน (Hallucinate)" ได้ง่ายเมื่อเจอโจทย์ที่ซับซ้อน เพราะมันไม่ได้ "คิด" ก่อนพูด

- **System 2 (Thinking Slow)**:
  - **ลักษณะ**: ช้า, ใช้ตรรกะ, มีการวางแผน, ต้องใช้ความพยายามและสมาธิสูง
  - **ตัวอย่างในมนุษย์**: การแก้สมการคณิตศาสตร์ 17 $\times$ 24, การจอดรถในที่แคบ, การเขียนโปรแกรม
  - **ตัวอย่างใน AI**: โมเดลยุคใหม่ (เช่น OpenAI o1) ที่มีกระบวนการ **Test-Time Compute** หรือการใช้เวลาคำนวณเพิ่มเติมระหว่างที่ถูกถาม โมเดลจะสร้าง Chain-of-Thought เพื่อวางแผน แยกแยะปัญหา ตรวจสอบข้อผิดพลาดของตัวเอง (Self-correction) และลองผิดลองถูกก่อนที่จะคายคำตอบสุดท้ายออกมา

วิวัฒนาการของ AI ในปัจจุบันคือการพยายามเปลี่ยนผ่านจากการทำงานแบบ System 1 ล้วนๆ ไปสู่การผสานรวม System 2 เข้ามาเพื่อแก้ปัญหาคณิตศาสตร์ โค้ดดิ้ง และตรรกะที่ซับซ้อนได้อย่างมนุษย์
