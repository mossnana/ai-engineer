# Forward Pass (พาสการคำนวณไปข้างหน้า)

[← กลับสู่เนื้อหาบทเรียน](../lectures/project2_rag_prompt_engineering.md)

**Forward Pass** (บางครั้งเรียกว่า Forward Propagation) คือกระบวนการป้อนข้อมูล (Input) เข้าไปในโครงข่ายประสาทเทียม แล้วปล่อยให้ข้อมูลนั้นเดินทางผ่านชั้นต่างๆ (Layers) ทะลุทะลวงจนถึงชั้นสุดท้ายเพื่อประเมินผลลัพธ์ (Output) หรือการคาดคะเน (Prediction)

ขั้นตอนนี้เป็นกระบวนการเดียวกับที่เราใช้ตอนนำโมเดลไปใช้งานจริง (Inference)

## คำอธิบายเชิงลึก (In-depth Explanation)
ในทางคณิตศาสตร์ Forward Pass คือชุดของการคำนวณเมทริกซ์เชิงเส้นตามด้วยฟังก์ชันกระตุ้น (Activation function) ที่ทำแบบเป็นทอดๆ 

สมมติมีโครงข่ายง่ายๆ 2 ชั้น:
1.  **Layer 1**: รับอินพุต $x$, คูณด้วยน้ำหนัก $W_1$, บวกค่าไบแอส $b_1$, ผ่านฟังก์ชัน $f_1$ 
    $$h_1 = f_1(W_1 x + b_1)$$
2.  **Layer 2**: รับอินพุตเป็น $h_1$ (ผลลัพธ์จากชั้นที่แล้ว), คูณด้วยน้ำหนัก $W_2$, บวกไบแอส $b_2$, ผ่านฟังก์ชัน $f_2$
    $$y = f_2(W_2 h_1 + b_2)$$

ผลลัพธ์สุดท้าย $y$ คือสิ่งที่เราต้องการ 

ในกระบวนการฝึกฝนโมเดล (Training) เราต้องทำ Forward Pass ก่อน เพื่อหาผลลัพธ์คาดคะเน $\hat{y}$ นำไปเทียบกับเฉลยจริง (Ground Truth) $y$ เพื่อหาค่าความสูญเสีย (Loss) จากนั้นจึงทำ **Backward Pass** (ย้อนศร) ค่อยๆ คำนวณอนุพันธ์กลับไปอัปเดตน้ำหนัก $W_1$ และ $W_2$ 

## ตัวอย่างการนำไปใช้ (Practical Example)
การประมวลผลโมเดลภาษา LLM ก็คือการทำ Forward Pass ครั้งใหญ่มากๆ ผ่านเลเยอร์ Transformer 80 ชั้น เมื่อใส่คำว่า "สวัสดี" เข้าไป ระบบจะนำเวกเตอร์ของคำว่า "สวัสดี" ค่อยๆ บวกลบคูณหารทะลุ 80 ชั้น จนได้ผลลัพธ์ออกมาเป็นคำว่า "ครับ/ค่ะ" ในตอนท้าย
